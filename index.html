Para que tu aplicación se pueda instalar como PWA en Android y agregar en iOS, necesitas: 1. **Manifest Web App** Agrega en ``: ```html ``` 2. **Archivo `manifest.json`** en la raíz: ```json { "name": "Calculadora Paralelo", "short_name": "Paralelo", "start_url": ".", "display": "standalone", "background_color": "#f7f7f8", "theme_color": "#3b82f6", "icons": [ { "src": "icons/icon-192.png", "sizes": "192x192", "type": "image/png" }, { "src": "icons/icon-512.png", "sizes": "512x512", "type": "image/png" } ] } ``` 3. **Service Worker básico** para cache offline (`service-worker.js`): ```javascript self.addEventListener('install', e => { e.waitUntil( caches.open('paralelo-cache').then(cache => { return cache.addAll([ '/', '/index.html', '/manifest.json', '/icons/icon-192.png', '/icons/icon-512.png' ]); }) ); }); self.addEventListener('fetch', e => { e.respondWith( caches.match(e.request).then(response => { return response || fetch(e.request); }) ); }); ``` 4. **Registrar el Service Worker** en tu JS principal: ```javascript if ('serviceWorker' in navigator) { navigator.serviceWorker.register('/service-worker.js') .then(() => console.log('SW registrado')) .catch(err => console.error('SW error', err)); } ``` Con esto, tu app será instalable en Android y agregable a la pantalla de inicio en iOS.
